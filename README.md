AUTOMATIC PET CARE SERVICE

Android app
built with Java and Firebase for managing pet care tasks.

demo link:https://drive.google.com/file/d/1n159-ViqHiX9dU9jBFg4bp7gETWdJ1sL/view?usp=sharing
APK LINK:https://drive.google.com/file/d/1XxSOcv_eTQ-VCWU5y5jFp_-ytgqgtNXx/view?usp=sharing


## Features
- User login/signup using Firebase Authentication
- Add multiple pets with name, breed, age, photo
- Set feeding & medication reminders with notifications
- Track vaccination dates and get alerts
- Save vet appointment history
- payment and payment details (history view)
- All data synced to Firebase Realtime Database


## Tech Stack
`Android Studio` `Java` `Firebase Auth` `Firebase Realtime DB` `XML`

## How to Run
1. Clone: `git clone https://github.com/SWETHA8685/Automatic-petcare-.git`
2. Open project in Android Studio
3. Add your `google-services.json` to `/app` folder from Firebase Console
4. Sync Gradle and run on emulator/physical device
